AI Emotion Detection System

Overview:

The AI Emotion Detection System is a real-time computer vision application that detects human faces through a webcam and predicts emotions using Artificial Intelligence.

The application captures live video, identifies faces, and classifies emotions such as Happy, Sad, Angry, Neutral, Fear, Surprise, and Disgust.

Features:

- Real-time webcam emotion detection
- Face detection
- Emotion classification
- Live emotion labels
- Easy-to-use interface

Technologies Used:

- Python
- OpenCV
- FER
- TensorFlow
- NumPy

Installation:

bash
pip install -r requirements.txt
